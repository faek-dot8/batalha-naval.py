# Batalha Naval em Python

Um jogo simples de Batalha Naval feito em Python no terminal.

## 📌 Sobre o Projeto

Este projeto foi criado para praticar lógica de programação, estruturas de repetição, condicionais e manipulação de listas em Python.

O jogador tenta encontrar os navios escondidos no mapa informando coordenadas.

## 🚀 Tecnologias Utilizadas

- Python 3
